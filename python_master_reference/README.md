# Python Master Reference

This folder contains 35 small Python files.

Each file focuses on one Python topic.

## How to run

Open Terminal in this folder and run for example:

```bash
python3 01_printing_output.py
```

or:

```bash
python3 28_oop_class_and_object.py
```

## Recommended learning method

1. Open one file.
2. Read the comments.
3. Run it.
4. Change some values.
5. Run it again.
6. Break something on purpose.
7. Fix it.

This is a good way to refresh Python syntax and OOP concepts.
