# Playwright Python Automation Project: OWASP Juice Shop

This project is a practical QA automation learning project using **Playwright with Python** and **pytest**.

The target application is **OWASP Juice Shop**, running locally with Docker.

The project focuses on real end-to-end testing scenarios such as login, signup, search, basket, checkout, form validation, and test refactoring.

---

## Project Goal

The goal of this project is to learn and demonstrate:

- Playwright automation with Python
- End-to-end web UI testing
- pytest test structure
- Locator strategies
- Assertions and validation checks
- Positive and negative test scenarios
- Reusable helper functions
- Test data management
- AI-assisted test development and debugging

---
